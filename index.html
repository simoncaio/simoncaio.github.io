POC HTML Smuggling - Cisco Secure Access

Fetch-based smuggling: busca EICAR de fonte externa, transforma via JS e reconstroi como .exe localmente.

Como funciona este teste:
O JavaScript faz fetch do arquivo EICAR diretamente de secure.eicar.org. O conteudo recebido e processado em memoria: aplicamos XOR para ofuscar, simulamos fragmentacao, revertemos a ofuscacao e reconstruimos o arquivo como .exe via Blob API com download automatico.

O Secure Access pode detectar em 2 pontos:
1) No fetch HTTP (SWG/antimalware bloqueia o EICAR na rede)
2) No download do Blob (RBI intercepta o arquivo reconstruido)
